Doodle.js

Overview:

This is the reference documentation for Doodle.js, a JavaScript drawing and animation library.

The manual provides information about the syntax and usage of supported elements within the framework. This includes classes, constants, types, parameters, and concise examples.

Certain areas of the documentation may be hidden from view by toggling the button on the right side. This should make it easier to navigate classes with many properties. To view these sections again, simply re-toggle the button.

For further usage information, it is recommended to look at the source code for examples located in the demos section. Additional questions may be asked on the Doodle.js mailing list.

This api and documentation is a work-in-progress. For the latest version, please update from the project's source repository.
